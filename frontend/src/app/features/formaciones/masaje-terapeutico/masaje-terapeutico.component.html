[ DESARROLLO POR DOMINGOS ]

Programa Académico

Sesión 1: Fundamentos Miofasciales

Principios anatómicos del tejido blando, palpación diagnóstica y técnicas básicas.

01 //

Sistema Fascial

Teoría Fascias

Anatomía del sistema fascial y Puntos de Gatillo (Trigger Points) en el cuerpo humano.

02 //

Maniobras Básicas

Práctica Técnicas

Deslizamiento profundo, amasamiento y fricción controlada en la musculatura superficial.

03 //

Palpación Diagnóstica

Evaluación Diagnóstico

Palpación y reconocimiento de bandas tensas en el tejido blando para localizar contracturas clínicas.

SPECIMEN // PALPACIÓN Y ANATOMÍA MIOFASCIAL

Sesión 2: Digitopresión Mecánica

Presión instrumental clínica, dosificación de fuerza y cuidado articular del terapeuta.

01 //

Instrumentación de Presión

Herramientas Madera

Uso de digitopresores profesionales de madera y resina para puntos gatillo difíciles.

02 //

Zonas de Tratamiento

Focalizado Espalda

Tratamiento de espalda alta, zona dorsal, trapecios y cervicales mediante presión y liberación fascial.

03 //

Ergonomía del Terapeuta

Postura Ergo

Ergonomía corporal y posicionamiento óptimo del terapeuta para evitar fatiga o lesiones en la consulta.

SPECIMEN // DIGITOPRESORES E INSTRUMENTOS

Sesión 3: Abordaje Clínico del Dolor

Protocolos de tratamiento para lumbalgias, cervicalgias y síndromes de tensión crónicos.

01 //

Protocolos para Contracturas

Tratamiento Lumbares

Protocolos clínicos estructurados para lumbalgias, contracturas severas y puntos gatillo complejos.

02 //

Estiramiento Asistido

Alivio Tracción

Técnicas de estiramiento muscular asistido y maniobras de tracción pos-tratamiento.

03 //

Contraindicaciones

Clínica Límites

Estudio de indicaciones clínicas, contraindicaciones absolutas y relativas para un tratamiento seguro.

SPECIMEN // MANIOBRAS DE TRACCIÓN Y ALIVIO

Sesión 4: Aplicación y Práctica Integrada

Casos prácticos de simulación integradora en el aula y feedback en tiempo real.

01 //

Prácticas en Camilla

Vivencial Camilla

Simulación clínica completa en camilla de clases aplicando todas las técnicas aprendidas.

02 //

Fichas de Evaluación

Registro Fichas

Evaluación inicial, seguimiento y evolución clínica del progreso y dolor del paciente.

03 //

Evaluación Supervisada

Examen Supervisión

Examen práctico final presencial supervisado por el docente para validación del aprendizaje.
